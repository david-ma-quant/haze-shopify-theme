## Spacing Scale

All values sourced from `assets/haze-tokens.css`. Use CSS custom properties, not raw values.

| Token | CSS Variable | Raw Value | Usage |
|-------|-------------|-----------|-------|
| xs | `--haze-space-xs` | 4px (0.4rem) | Icon gaps, inline element padding |
| sm | `--haze-space-sm` | 8px (0.8rem) | Card internal padding, button padding-x |
| md | `--haze-space-md` | 16px (1.6rem) | Default element spacing, grid gaps on mobile |
| lg | `--haze-space-lg` | 32px (3.2rem) | Section internal padding, grid gaps on desktop |
| xl | `--haze-space-xl` | 64px (6.4rem) | Section vertical padding (hero, bottom CTA) |

Exceptions:
- Hero section uses `100vh` / `100dvh` for height — not a spacing token (CONTEXT.md D-02)
- Bottom CTA vertical padding: `--haze-space-lg` (32px) each side — intentionally smaller than hero (CONTEXT.md D-22)
- Touch targets (CTA button, card hover area): minimum 44px height to meet WCAG 2.5.5

Source: haze-tokens.css, CONTEXT.md D-07 (spacing scale), D-02 (hero height), D-22 (bottom CTA)

## Typography

### Type Scale

4 sizes maximum. 2 weights maximum. All component text must map to one of these 4 rows exactly — no additional pixel values permitted anywhere in the implementation.

| Role | Font Family | Size | Weight | Line Height | Letter Spacing | Notes |
|------|------------|------|--------|-------------|----------------|-------|
| Display | Cormorant Garamond | 56px desktop / 36px mobile | 300 | 1.1 | 0 | Italic variant for hero headline |
| Heading | Cormorant Garamond | 24px (1.5rem) | 400 | 1.2 | 0 | All section and card headings |
| Body | DM Sans | 16px (1rem) | 400 | 1.5 | 0 | Body copy, descriptions, testimonial quotes |
| Label | DM Sans | 13px (0.8125rem) | 400 | 1.3 | 0.05em | Button text, reviewer names, nav links, captions |

Weight system: **300** (editorial italic accent — Display role only) and **400** (all other roles). No third weight is permitted. Differentiate elements within a role using letter-spacing, opacity, italic style, and uppercase transform — never by adding a new weight or pixel size.

### Per-Component Typography Assignments

Each element maps to exactly one row in the Type Scale above. No size or weight overrides permitted.

| Component | Element | Role | Differentiation |
|-----------|---------|------|----------------|
| Hero | "Stay tender." headline | Display | italic, weight 300 |
| Hero | Subtext | Body | opacity 0.8 |
| Hero | CTA button | Label | uppercase, letter-spacing 0.12em |
| Product grid card | Kit component name | Heading | — |
| Product grid card | Short description | Body | opacity 0.75 |
| Value prop | Block heading | Heading | — |
| Value prop | Block body | Body | opacity 0.8 |
| Social proof | Testimonial quote | Body | italic |
| Social proof | Reviewer name | Label | uppercase, letter-spacing 0.08em, opacity 0.6 |
| Bottom CTA | Headline | Heading | — |
| Bottom CTA | Subtext | Body | opacity 0.75 |
| Bottom CTA | CTA button | Label | uppercase, letter-spacing 0.12em |
| Nav links | Nav link | Label | uppercase, letter-spacing 0.1em (Phase 1 D-14) |

Source: CONTEXT.md D-02 (hero), D-08 (card), D-15/16 (value props), D-19 (social proof), D-22 (bottom CTA), Phase 1 D-14 (nav)

## Component Inventory

### Section: haze-hero.liquid

**Layout:**
- Full-viewport height: `height: 100vh; height: 100dvh` (100dvh preferred for mobile browser chrome)
- Content centered both axes via CSS Flexbox (`display: flex; align-items: center; justify-content: center`)
- Text content centered, max-width 600px on desktop
- Background: `object-fit: cover` full-bleed image (`haze-placeholder-hero.jpg`)
- Overlay: semi-transparent dark scrim `rgba(44, 36, 32, 0.25)` over image for text legibility

**Elements:**
- Headline: "Stay tender." — Display size (56px desktop / 36px mobile), Cormorant Garamond weight 300, italic
- Subtext: 1 line, Body size (16px), DM Sans weight 400, opacity 0.8, white
- CTA button: "Shop the Kit" — Label size (13px), links to `/products/hair-ritual-kit`
- Button style: terracotta fill, cream text, `--haze-radius-button: 2px`, min-height 44px, padding 12px 32px, uppercase, letter-spacing 0.12em

**States:**
- Default: static placeholder image visible
- After real assets: swap `haze-placeholder-hero.jpg` for product photography

### Section: haze-product-grid.liquid + Snippet: haze-card-video.liquid

**Grid layout:**
- Desktop (990px+): 3 columns, `gap: var(--haze-space-lg)` (32px)
- Tablet (750px–989px): 2 columns, `gap: var(--haze-space-md)` (16px)
- Mobile (<750px): 1 column, `gap: var(--haze-space-md)` (16px)
- Fifth card: `grid-column: auto` — flows naturally; on 3-column desktop it sits in position 5 (col 2), centered via justify-items or text-align on content
- Section padding: `var(--haze-space-xl)` (64px) top/bottom, `var(--haze-space-lg)` (32px) left/right

**Card anatomy (`haze-card-video.liquid`):**
- Aspect ratio: 3/4 (portrait) for image/video area
- `--haze-radius-card: 4px` applied to card container
- Image: `object-fit: cover`, fills card image area, lazy loading
- Overlay: 0.3s opacity crossfade between static image and video on hover (CONTEXT.md D-11)
- Kit component name: Heading role (24px), Cormorant Garamond weight 400, below image
- Short description: Body role (16px), DM Sans weight 400, opacity 0.75, below name
- No price, no ATC button (showcase only — CONTEXT.md D-08)
- Card background: `--haze-color-cream`

**Video states (`haze-card-video.liquid`):**
- No video URL set: renders `<img>` only, no `<video>` in DOM (VIDEO-01)
- Video URL set: renders `<video muted playsinline loop preload="none">` with `<source type="video/mp4">`
  - mp4 is the only source format (VIDEO-05 confirms single source)
- Hover state (desktop): `mouseenter` → video.play(), `mouseleave` → video.pause() (VIDEO-02)
- Touch/mobile state: IntersectionObserver → autoplay when ≥50% in viewport, pause when out (VIDEO-03)
- Reduced motion: `@media (prefers-reduced-motion: reduce)` → no video rendered, static image only (VIDEO-04)
- Off-screen pause: IntersectionObserver threshold 0 → pause when card leaves viewport (VIDEO-02)
- Transition: `opacity: 0` to `opacity: 1` on video, 0.3s ease — image remains behind (D-11)

**Card placeholder image naming (CONTEXT.md D-09):**
- `haze-placeholder-silk-comb.jpg`
- `haze-placeholder-brush.jpg`
- `haze-placeholder-towel.jpg`
- `haze-placeholder-scrunchie.jpg`
- `haze-placeholder-gift-box.jpg`

## Interaction Contract

| Interaction | Trigger | Response | Duration |
|-------------|---------|----------|----------|
| Hero CTA click | Tap/click | Navigate to `/products/hair-ritual-kit` | Immediate |
| Card video start | `mouseenter` on desktop | `video.play()`, image fades to `opacity: 0` | 300ms ease |
| Card video pause | `mouseleave` on desktop | `video.pause()`, image returns to `opacity: 1` | 300ms ease |
| Card video autoplay | 50%+ card in viewport (touch) | `video.play()` | Immediate |
| Card video off-screen pause | Card exits viewport | `video.pause()` | Immediate |
| Reduced motion | `prefers-reduced-motion: reduce` | No video rendered, static images throughout | Not applicable |
| Bottom CTA click | Tap/click | Navigate to `/products/hair-ritual-kit` | Immediate |

---

## Accessibility Contract

- All `<img>` elements must have descriptive `alt` text (kit component name + "placeholder")
- `<video>` elements must have `aria-hidden="true"` (decorative, not content)
- CTA buttons must have minimum 44px height (WCAG 2.5.5 target size)
- Color contrast: dark `#2C2420` on cream `#F7F3EE` = ~12:1 (AAA), dark on blush `#E8D5CE` = ~8:1 (AAA)
- Cream text on terracotta button `#C4714A` — verify minimum 4.5:1 for body-size text; use `#FFFFFF` if fails
- `prefers-reduced-motion` handled at CSS and JS layer (VIDEO-04)
- Social proof section: when hidden, no DOM nodes output (not `display:none` hiding)
- Focus ring: inherit Dawn's default focus styles — do not override
