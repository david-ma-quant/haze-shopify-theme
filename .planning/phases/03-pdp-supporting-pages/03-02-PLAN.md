**Create `assets/haze-pages.css`** — shared CSS for both About and FAQ pages (per D-21, D-25).

This file loads on both `page.about.json` and `page.faq.json` templates. It must override Dawn's default typography with Haze brand fonts without breaking Dawn's responsive layout.

**IMPORTANT:** This CSS file cannot be loaded automatically from a JSON template. It must be loaded from within a section. The approach: Dawn sections that are used in these templates will need the CSS injected. Since we cannot modify Dawn sections, we need an alternative approach.

**Solution:** Create a minimal custom Liquid snippet or use a `custom_liquid` block approach. However, the simplest Shopify approach is to load the CSS from `layout/theme.liquid` with a page-type check. But the CLAUDE.md convention says CSS-only stylesheet blocks may be appended to Dawn sections when no alternative exists.

**Best approach for v1:** Add the stylesheet_tag to `layout/theme.liquid` with a conditional check. Read `layout/theme.liquid` first to find the right insertion point (after the existing `haze-tokens.css` tag).

Actually, the cleanest approach: load `haze-pages.css` from the `page.about.json` and `page.faq.json` templates using a custom Liquid section that only loads the CSS. But we should avoid creating extra sections just for CSS loading.

**Simplest correct approach:** Add `{{ 'haze-pages.css' | asset_url | stylesheet_tag }}` to `layout/theme.liquid` inside a conditional:

```liquid
{%- if template.suffix == 'about' or template.suffix == 'faq' -%}
  {{ 'haze-pages.css' | asset_url | stylesheet_tag }}
{%- endif -%}
```

This goes in the `<head>` section of `layout/theme.liquid`, after the existing `haze-tokens.css` tag. Read `layout/theme.liquid` to find the exact insertion point.

```css
/* Haze Pages — shared CSS for About + FAQ page brand overrides
   Loaded conditionally on page.about and page.faq templates */

/* ========================================
   ABOUT PAGE — image-banner hero overrides
   ======================================== */

/* Hero headline — Display role: Cormorant Garamond italic */
.banner__heading {
  font-family: var(--font-heading-family);
  font-weight: 300;
  font-style: italic;
  line-height: 1.1;
}

/* Hero subtext — Body role */
.banner__text {
  font-family: var(--font-body-family);
  font-weight: 400;
  font-size: 1rem;
  line-height: 1.5;
}

/* ========================================
   ABOUT PAGE — image-with-text overrides
   ======================================== */

/* Section headings — Heading role */
.image-with-text h2,
.image-with-text .image-with-text__heading {
  font-family: var(--font-heading-family);
  font-size: 1.5rem;
  font-weight: 400;
  line-height: 1.2;
  color: var(--haze-color-dark);
}

/* Body text — Body role */
.image-with-text .image-with-text__text p {
  font-family: var(--font-body-family);
  font-size: 1rem;
  font-weight: 400;
  line-height: 1.5;
  color: var(--haze-color-dark);
  opacity: 0.85;
}

/* ========================================
   FAQ PAGE — collapsible-content overrides
   ======================================== */

/* Section heading — Heading role */
.collapsible-content .collapsible-content__heading {
  font-family: var(--font-heading-family);
  font-size: 1.5rem;
  font-weight: 400;
  line-height: 1.2;
  color: var(--haze-color-dark);
}

/* Question text (summary) — Heading role */
.collapsible-content summary .h4,
.collapsible-content summary h2 {
  font-family: var(--font-heading-family);
  font-size: 1.5rem;
  font-weight: 400;
  line-height: 1.2;
  color: var(--haze-color-dark);
}

/* Answer text (details body) — Body role */
.collapsible-content .collapsible-content__panel .rte,
.collapsible-content .collapsible-content__panel p {
  font-family: var(--font-body-family);
  font-size: 1rem;
  font-weight: 400;
  line-height: 1.5;
  color: var(--haze-color-dark);

}

/* Accordion item separator — subtle bottom border */
.collapsible-content .collapsible-content__panel {
  border-color: rgba(44, 36, 32, 0.15);
}

/* Summary padding */
.collapsible-content summary {
  padding: var(--haze-space-sm) 0;
}
```

Typography mapping (from UI-SPEC):
- About hero headline: Display role — Cormorant Garamond, weight 300, italic, line-height 1.1 (Dawn handles the responsive sizing via heading_size "h0")
- About editorial heading: Heading role — 24px (1.5rem), Cormorant Garamond, weight 400
- About body copy: Body role — 16px (1rem), DM Sans, weight 400, opacity 0.85
- FAQ section heading: Heading role — 24px (1.5rem), Cormorant Garamond, weight 400
- FAQ question (summary): Heading role — 24px (1.5rem), Cormorant Garamond, weight 400
- FAQ answer: Body role — 16px (1rem), DM Sans, weight 400, opacity 0.85
- Accordion border: `rgba(44, 36, 32, 0.15)` — the dark color at 15% opacity (cannot use CSS custom property with alpha)

No hex values in CSS except `rgba(44, 36, 32, 0.15)` for the border (requires alpha channel).
